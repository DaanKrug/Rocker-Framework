The Rocker Framework

Let's Rock! Comm'on be a Rock'star!

       Yeah, coding whit Rocker you can be a Rocker Star! (if you wish)

       Rocker is a pure Javascript framework to make easy the data forms creation, making transparent to programmers the repetitive phases of CRUD operations construction. Better explaining: you don't need bother in take data from inputs, then construct the AJAX submission methods of data, nor the reverse way getting received data from server and manually putting into respective input fields.
As well is projected whit Tableless concept, and the <table> HTML tag is not present in any generated HTML structures.
Another projected feature is the smart server-side data pagination. Whit this concept, only the rows of actual page ( on table data ) are loaded from server to be draw in user interface. The back-end no need make cache storage of the loaded data.
This implementation solution provide a minimal use of resources on front-end and back-end.

       You need only construct the form data, using a RockerForm object, and then go to back-end write methods that handle the requests on server. The bridge between the user interface and the application server is work for Rocker Framework.
The RockerForm already uses internally a RockerTable embedded. However its possible use a RockerTable whitout relation whit any RockerForm.
All these possibilities are demonstrated in "Rocker Framework Show Case" tab, and all the source code files of Show Case are avaliable in "Show Case Source Code" tab.

       Rocker framework mechanism was totally coded from zero. However uses visual themes from JQuery UI and Bootstrap.
The TextArea component makes use from a integrated ckEditor component, when is draw in rich editor mode.
JQuery is a Bootstrap required dependency.

 

Third parts libraries (contained in "vendors" folder)

       bootstrap-3.3.7-dist (https://getbootstrap.com/): Provides css styles for visual effects, responsive layout/components and component aligning.
Some uses in:
- navigation bars (.navbar .navbar-default),
- container (.container, .container-fluid),
- component aligning (.col-xs-1 to .col-xs-12, .col-sm-1 to .col-sm-12)

       jquery-3.2.1 (http://jquery.com/): Dependency required by bootstrap library.

       jquery-ui-1.12.1 (http://api.jqueryui.com/): Provides css styles for containers, headers, inputs, icons, buttons, and others.
Some uses in:
- buttons/inputs (.ui-button .ui-corner-all .ui-icon )
- containers/headers (.ui-widget .ui-widget-content .ui-widget-header .ui-corner-all .ui-corner-top)

       CKEditor 4 (http://ckeditor.com): TextArea element was build to interact whit ckeditor and use rich editor provided by them. All transparently too. Its no required know how to use the ckeditor, to creates rich editor data inputs.
